###使用命令

|  命令行           |    含义      |
| ---------------- | -------------|
| git clone HTTPS  | 克隆仓库      |
| git add .        | 添加到暂存区  |
| git commit -m '备注语句' |  添加到本地仓库 |
| git push         | 添加到github上 |
| git checkout -b 新分支名称 | 添加新分支并且切换到新分支上 |
| git merge 新分支名称 | 合并新分支到主分支上 |
| git push -u origin 新分支名称 | 本地分支推送到云端 |
| git branch | 查看分支 |
| git status | 查看工作区 |

注意合并分支时先切换到主分支上进行操作，合并后要使用push语句更新


###结构
工作区 暂存区 本地仓库 远程仓库
